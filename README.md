My neovim configuration
=============================

## Installation (neovim)
<pre>
  rm ~/.config/nvim -rf
  git clone git@github.com:ArthurSonzogni/neovim-configuration.git ~/.config/nvim
</pre>


## Post installation
* :PlugInstall in vim
* install patched font for [airline](https://github.com/vim-airline/vim-airline)
