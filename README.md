My neovim / vim configuration
=============================

## Installation (neovim)
<pre>
  rm ~/config/nvim
  git clone git@github.com:ArthurSonzogni/neovim-configuration.git ~/config/nvim
</pre>

## Installation (vim)
<pre>
  rm ~/.vim
  git clone git@github.com:ArthurSonzogni/neovim-configuration.git ~/.vim
</pre>

## Post installation
type : : PlugInstall in vim
