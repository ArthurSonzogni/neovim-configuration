My neovim configuration
=============================

## Installation (neovim)
<pre>
  rm ~/.config/nvim -rf
  git clone http://github.com/ArthurSonzogni/neovim-configuration.git ~/.config/nvim
</pre>


## Post installation
* :PlugInstall in nvim
* install patched font for [airline](https://github.com/vim-airline/vim-airline)
